# aux Documentation

This folder contains documentation for the aux skills collection.

## Skills

| File | Description |
| ---- | ----------- |
| [GREP.md](GREP.md) | Agent-assisted text search using ripgrep. Find patterns across codebases with natural language intent. |
| [FIND.md](FIND.md) | Agent-assisted file enumeration using fd. Discover files and directories by name, extension, or pattern. |
| [DIFF.md](DIFF.md) | Deterministic git diff inspection. Bounded change summaries with optional patch output. |
| [LS.md](LS.md) | Deterministic directory state inspection. Inventory, ranking, and optional git status mapping. |

## Quick Reference

### When to use each skill

- **grep** — "Where is X used?" / "Find all occurrences of Y"
- **find** — "What files match this pattern?" / "List all config files"
- **diff** — "What changed?" / "Show me the staged changes"
- **ls** — "What's in this directory?" / "Show largest files" / "What was modified recently?"

### Common patterns

```bash
# Text search
scripts/skill.sh run --root /path/to/repo --pattern "oauth2" --mode fixed

# File enumeration
scripts/skill.sh run --root /path/to/repo --extension py --type file

# Git diff summary
scripts/skill.sh run --compare working_vs_index

# Directory inventory
scripts/skill.sh run --root /path/to/dir --order-by size --direction desc
```
