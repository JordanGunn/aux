# Triggers

## Activation

This skill activates when the user invokes `/find` followed by a natural language prompt describing what files they want to find.

## Examples

- `/find find all Python files in the project`
- `/find list configuration files`
- `/find show me all markdown documentation`
- `/find find test files`
- `/find enumerate all scripts in the repo`

The prompt is treated as intent. The agent compiles it into explicit enumeration parameters.
